# Product Overview

## What The Dashboard Is

The HomeTrust Africa dashboard is a secure client operations portal for Africans in the diaspora managing projects in Nigeria and, over time, other African markets. It is not a public marketplace and it is not a self-serve consumer SaaS product. Access is provisioned internally by HomeTrust Africa.

## Core Product Purpose

The dashboard exists to make high-trust remote execution feel transparent, controlled, and auditable. Clients use it to:

- track active projects and milestones
- review reports, files, receipts, and media
- monitor budget and spend visibility
- respond to approvals
- message the HomeTrust Africa team securely
- manage account settings and communication preferences
- request support or escalation

## Supported Project Categories

The portal is intentionally broader than home construction. Seeded and modeled project types include:

- construction
- business setup
- agriculture
- solar / infrastructure
- land development
- procurement / logistics

## Target Users

### Client

Primary external role. Clients see only their own projects and client-safe artifacts.

### Team Member

Internal delivery role assigned to specific projects. Team members can access scoped project data and operational workflows where the implementation exposes them.

### Operations Manager / Admin

Internal oversight roles with broader access, invite foundations, and audit visibility.

## Major User Workflows

### Client workflows

- accept an invite and activate access
- sign in and land in `/dashboard`
- review project updates, reports, budgets, files, and conversations
- respond to pending approvals
- adjust profile, security, and notification settings
- open support requests and follow up in support threads

### Internal workflows already supported in code

- provision an invite through a secure server action foundation
- publish dashboard-visible records through the Supabase-backed data model
- receive notifications when clients reply to support or conversations

## Product Value Proposition

The portal is designed to reduce uncertainty. It gives diaspora clients:

- visibility into what is happening
- traceability for why it happened
- secure access to supporting evidence
- a clear line of communication with the team
- enough operational structure to trust the platform with real money and real-world execution
