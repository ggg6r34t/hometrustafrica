# Feature: Settings

## Purpose

Settings gives clients control over their identity, password, notification preferences, and workspace preferences.

## Routes

- `/dashboard/settings/profile`
- `/dashboard/settings/security`
- `/dashboard/settings/notifications`
- `/dashboard/settings/preferences`

## Profile

- full name
- email
- phone
- diaspora country
- preferred contact method

## Security

- password rotation
- last sign-in display
- two-factor readiness summary
- active sessions count summary

## Notifications

- email alerts
- in-app alerts
- report, milestone, budget, and message toggles

## Preferences

- timezone
- currency
- density

## Honest Notes

- Active sessions are still summarized rather than fully managed
- Two-factor support is represented as readiness metadata, not a full enrollment flow
